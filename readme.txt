# Sociotechnical Autonomy Analysis

Code for "The Emergence of Sociotechnical Autonomy" (Nature Communications, 2025).

## Requirements
- Python 3.10+
- ~200GB disk space (for cloning repositories)
- See `requirements.txt`

## Reproduction
1. Clone repositories listed in `data/repos_config.json`
2. Update paths in `analysis.py` (BASE_PATH variable)
3. Run: `python analysis.py`
